# Security Policy

## Supported Versions

Solar Earth Watch is currently in an initial public version. Security fixes apply to the latest commit on the default branch.

## Reporting a Vulnerability

Please open a GitHub security advisory or private report if available for the repository. If that is not available, open an issue with non-sensitive reproduction details and avoid posting secrets, tokens, or private data.

## Data and API Use

This project uses public NOAA, NASA/CCMC, and Helioviewer endpoints and does not require authentication or paid API keys. Do not add credentials to source code, fixtures, screenshots, or issue reports.
