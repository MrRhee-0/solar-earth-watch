# Notice

Solar Earth Watch uses public data and imagery interfaces from the following sources:

- NOAA Space Weather Prediction Center (SWPC) solar wind plasma, magnetometer, and planetary K-index products.
- NASA/CCMC DONKI web services for solar and heliospheric event records.
- Helioviewer Project API for solar imagery access.

Data and imagery remain the work of their respective providers. This project uses those sources as witness surfaces for a downstream public dashboard and does not represent an official NOAA, NASA, CCMC, or Helioviewer product.

This project is not an operational forecasting product and must not be used for safety-critical decisions.
